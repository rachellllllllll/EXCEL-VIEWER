# Excel Viewer

פרויקט פשוט להצגת 10 שורות ראשונות × 10 עמודות מקובץ Excel.

## התקנה

```bash
npm install
```

## הרצה מקומית

```bash
npm run dev
```

## בנייה לפרודקשן

```bash
npm run build
npm run preview
```

## העלאה ל-Vercel

1. התחבר ל-Vercel
2. הוסף את הפרויקט
3. Vercel יזהה אוטומטית Vite ויבנה את הפרויקט

הפרויקט כולל את כל ה-polyfills הנדרשים ל-xlsx לעבוד ב-Vercel!

## טכנולוגיות

- React 18
- TypeScript
- Vite 4
- xlsx (עם polyfills מלאים)
- Buffer, Process, Stream polyfills
